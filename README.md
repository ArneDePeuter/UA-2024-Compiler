# Compilers

```bash
java -jar ../../antlr-4.13.1-complete.jar -Dlanguage=Python3 MyGrammar.g4 -visitor   
```

```bash
python -m src.main --input  example_source_files/<proj1/proj2/proj3>_<man/opt>_<pass/syntaxErr/semanticErr>_<explanation>.c
```
